# GPT-Image-2 提示词精选集

精选优质 `gpt-image-2` 提示词，主打可直接复制、稍改即用、持续更新。

这个仓库暂时不按题材分组，先把真正值得收藏的内容沉淀下来。等精选数量足够，再做更细的人像、海报、电商、动漫、UI、改图分类。

## 这个仓库的定位

- 只放值得收藏的 Prompt
- 优先收录已经验证过传播效果的案例
- 保留原始提示词，同时补充中文理解
- 先做精选集，再做细分类

## 收录标准

- 能直接复制使用
- 主体、风格、构图、光线表达明确
- 有传播验证或强复用价值
- 适合继续二次改写
- 不收录明显低质、擦边或平台风险内容
